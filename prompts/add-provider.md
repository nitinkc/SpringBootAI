# Prompt: Add New LLM Provider\n\n## Purpose\nStep-by-step guide to integrate a new LLM provider (OpenAI, Claude, Gemini, etc.)\n\n## When to Use\nWhen you want to:\n- Add support for a different LLM vendor\n- Switch providers for cost/quality reasons\n- Offer multi-provider load balancing\n- Test new models\n\n## Prerequisites\n1. Read: `.github/skills/llm-integration/SKILL.md`\n2. Have API key/credentials for new provider\n3. Understand existing AiClient interface\n\n## Step 1: Create Provider Implementation\n\n**File**: `src/main/java/com/example/aispring/client/[ProviderName]AiClient.java`\n\nFollows this template:\n```java\n@Service\npublic class [Provider]AiClient implements AiClient {\n    \n    private final RestTemplate restTemplate;\n    private final [Provider]Config config;\n    \n    public [Provider]AiClient(\n        RestTemplate restTemplate,\n        [Provider]Config config) {\n        this.restTemplate = restTemplate;\n        this.config = config;\n    }\n    \n    @Override\n    public String generateResponse(String prompt) {\n        // Make API call\n        // Handle timeouts\n        // Parse response\n        // Track cost\n        // Return result\n    }\n    \n    @Override\n    public String generateResponseWithContext(String context, String prompt) {\n        // Combine context + prompt\n        // Call API\n        // Return response\n    }\n    \n    @Override\n    public boolean isAvailable() {\n        // Check if provider is healthy\n        // Return true/false\n    }\n    \n    @Override\n    public String getModelName() {\n        return \"[provider-model-name]\";\n    }\n}\n```\n\n## Step 2: Create Configuration\n\n**File**: `src/main/java/com/example/aispring/config/[Provider]Config.java`\n\nTemplate:\n```java\n@Configuration\n@ConfigurationProperties(prefix = \"ai.providers.[provider]\")\n@Data\npublic class [Provider]Config {\n    private String apiKey;\n    private String baseUrl;\n    private String modelName;\n    private Integer timeout;  // milliseconds\n    private Double temperature;  // 0.0 to 1.0\n    private Integer maxTokens;\n    \n    @Bean\n    @ConditionalOnProperty(name = \"ai.provider\", havingValue = \"[provider]\")\n    public AiClient [provider]Client(RestTemplate restTemplate) {\n        return new [Provider]AiClient(restTemplate, this);\n    }\n}\n```\n\n## Step 3: Add Configuration to application.yml\n\n```yaml\nai:\n  provider: [provider]  # or use load balancing\n  providers:\n    [provider]:\n      apiKey: ${AI_PROVIDER_KEY}\n      baseUrl: https://api.provider.com\n      modelName: model-name-v1\n      timeout: 10000\n      temperature: 0.7\n      maxTokens: 2000\n```\n\n## Step 4: Add Dependency to pom.xml\n\nIf provider has official SDK:\n```xml\n<dependency>\n    <groupId>com.provider</groupId>\n    <artifactId>provider-sdk</artifactId>\n    <version>1.0.0</version>\n</dependency>\n```\n\nOtherwise:\n```xml\n<!-- Already have RestTemplate from Spring Web -->\n```\n\n## Step 5: Implement Cost Tracking\n\n```java\nprivate double calculateTokenCost(int inputTokens, int outputTokens) {\n    // Provider-specific pricing\n    double inputCost = inputTokens * config.getInputTokenPrice();\n    double outputCost = outputTokens * config.getOutputTokenPrice();\n    return inputCost + outputCost;\n}\n```\n\n## Step 6: Write Tests\n\n**File**: `src/test/java/com/example/aispring/client/[Provider]AiClientTest.java`\n\n```java\n@SpringBootTest\nclass [Provider]AiClientTest {\n    \n    @MockBean\n    private RestTemplate restTemplate;\n    \n    @Autowired\n    private [Provider]AiClient client;\n    \n    @Test\n    void testGenerateResponse() {\n        // Mock API response\n        when(restTemplate.postForObject(anyString(), any(), any()))\n            .thenReturn(createMockResponse());\n        \n        // Call client\n        String result = client.generateResponse(\"test\");\n        \n        // Assert\n        assertNotNull(result);\n        assertTrue(result.length() > 0);\n    }\n    \n    @Test\n    void testTimeout() {\n        when(restTemplate.postForObject(anyString(), any(), any()))\n            .thenThrow(new ResourceAccessException(\"Timeout\"));\n        \n        assertThrows(TimeoutException.class, \n            () -> client.generateResponse(\"test\"));\n    }\n}\n```\n\n## Step 7: Try
